How to Get API Keys

Facebook & Instagram:

  1. Go to developers.facebook.com and create a Meta Developer account
  2. Create a new App (type: Business)
  3. Add the "Pages API" and "Instagram Graph API" products
  4. Generate a Page Access Token with pages_manage_posts permission
  5. For Instagram, link your Instagram Business account to your Facebook Page

TikTok:

  1. Go to developers.tiktok.com and create a Developer account
  2. Create a new App and request the "Content Posting API" scope
  3. After approval, generate your Access Token
